# **MovieRecommenderApp**

A **Movie Recommender System** built with **Python**, **scikit-learn**, and **Streamlit** that provides personalized movie recommendations based on user preferences.

---

## **Features**

- Recommend movies similar to your favorites  
- Interactive web interface powered by **Streamlit**  
- Content-based filtering using **scikit-learn**  
- Easy to use and deploy

## **Installation**

1. Clone the repository:

   
```bash
git clone https://github.com/Jasbe04/MovieRecommenderApp.git
```
2. Navigate to the project folder:


```
bash
cd MovieRecommenderApp
```
3.Install dependencies:

```
bash
Copy code
pip install -r requirements.txt
```
Usage

1. Run the Streamlit app:
   
```
bash
streamlit run app.py
```
2. Open the URL provided in the terminal to use the app.

3. Select your favorite movie or enter a movie name to get recommendations.

## **Technologies Used**

- Python – programming language

- scikit-learn – for content-based recommendation

- Streamlit – for web interface

- Pandas & NumPy – for data handling
